Fred's Farm

C++, WxWidgets, BeagleBone Black

This is a game where hopeless preys have to survive against courageous, savage predators. Predators and preys are separated from one another via obstacles, and it is the goal of the preys to survive to predators until they starve and for predators to eat all of the prey.

Fred's farm is written in C++ for the BeagleBone Black using WxWidgets. This makes the game cross-platform across Desktop devices.

BioBrace VR - A Multifunction Sensor for Gaming

ARM Mbed, C++, Unreal Engine, Swift, iOS

BioBrace VR is a wearable that is able to interact with mobile and desktop applications through USB and Bluetooth low energy (BLE) protocols to provide inputs and actions, based on the user's position, motion, muscle tension, and stress levels.

This project served as both a project conducted at the OpenHID Lab for future gesture elicitation experiments and as my Senior Design capstone project. I'd like to thank Omar Gonzalez, Elias Figueroa, Charlotte Farolan, and Ricardo Arbelaez. Without their awesome support, this project never would've shed light.

The firmware for BioBrace VR is written in C++, using the ARM Mbed framework. There is a mobile application component written in Swift and a puzzle ball game written using the Unreal Engine.

FPGA Greenhouse Temperature Controller

Xilinx ISE, C++

This project encompasses a simulation of a temperature controller using a Xilinx FPGA development board. The system stores the optimal conditions at which certain types of plants are expected to grow and regulates the temperature to meet the ideal temperature standards.

This project was built using the Xilinx ISE development ecosystem in C++.

Spatial Augmented Reality Environment

Unity, C#, Microsoft HoloLens

This project simulates outer space on the Microsoft HoloLens Augmented Reality headset. Its purpose is to serve as a gesture elicitation experiment to collect user gestures for the HoloLens, in order to determine what gestures work best for rotation and translation tasks.

The Unity Engine and C# were used to develop this project.
